# 项目简介

## 核心功能
- 基于YOLOv8的办公室人员活动检测系统
- 支持6类目标检测：chair occupied、chair vacant、moving person、occupied desk、sitting person、vacant desk
- 提供900张标注数据集
- 支持图片、视频及摄像头三种输入方式进行检测
- 提供完整UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 可选择不同检测模型
- 包含完整程序文件、UI界面源文件、图标、数据集图片及运行教程

## 适用场景
- 办公室人员活动状态监测
- 办公空间使用情况分析
- 人员行为研究
- 办公环境自动化管理

## 技术特点
- 基于Pycharm+Anaconda或VSCode+Anaconda开发环境
- 使用Python 3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 集成YOLOv8目标检测模型
- 提供图形化用户界面，操作便捷
- 已完成模型训练，配置环境后可直接使用
